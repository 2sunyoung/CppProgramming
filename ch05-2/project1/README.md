# ch05-2 실습과제1

## 문제1
참조에 의한 호출과 주소에 의한 호출의 차이점을 설명하라.

## 답
참조에 의한 호출은 인자에 대한 별명만 사용하는 개념이라 매개변수를 메모리에 할당하는 것과 인자값을 매개변수에 복사하는 과정이 필요 없다.<br>
하지만, 주소에 의한 호출은 포인터 변수가 매개변수로 할당되고, 인자값을 매개변수에 복사하는 과정이 있다.<br>
또 사용법에서도 차이가 있는데, 참조에 의한 호출은 주소에 의한 호출과 달리 주소 연산자(&)나 간접 참조연산자(*)와 같은 연산자를 사용하지 않는다.<br>

## 문제2
참조에 의한 호출의 장점은 무엇인가?

## 답
참조에 의한 호출은 주소에 의한 호출을 써야하는 경우(값 변경, 배열 데이터 전달, 여러개의 값 리턴 등)에 <br>
&나 *과 같은 연산자 사용 없이 간편하게 별명을 만들어 직관적인 코드를 짤 수 있다는 장점이 있다.<br>
