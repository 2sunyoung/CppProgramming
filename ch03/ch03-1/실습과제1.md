# ch03-1 실습과제1

## 문제1
구조체와 클래스의 차이점을 설명하라.

## 답
구조체는 관련있는 멤버 변수만을 모아 놓은 자료형이지만, 클래스는 관련있는 멤버 변수에 멤버 함수까지 모아 놓을 수 있는 자료형이다.

## 문제2
클래스와 객체의 차이점을 설명하라.

## 답
클래스는 사용자 정의 '자료형'이고 객체는 클래스 타입의 '변수'이다.

## 문제3
sizeof(클래스명)의 결과값을 출력해보고 결과를 설명하라.

## 답
sizeof(클래스명)은 멤버 변수들의 크기와 메모리 정렬을 위한 패딩을 포함한 클래스 변수의 메모리 크기를 반환한다.<br><br>
이때 메모리 정렬이란 CPU가 데이터를 읽는 속도를 높이기 위해 각 자료형의 크기에 맞는 배수 주소에 데이터를 배치하는 것을 말한다.<br>
또, 패딩이란 앞서 말한 메모리 정렬을 위해 컴파일러가 자동으로 넣는 빈 공간이라 할 수 있다.

예시) <br>
class Size{<br>
public:<br>
⠀⠀⠀char a;<br>
⠀⠀⠀double b;<br>
⠀⠀⠀char c;<br>
⠀⠀⠀double X();<br>
};<br><br>
위와 같은 상황은 <br><br>
a⠀⠀⠀⠀⠀⠀1<br>
패딩⠀⠀⠀⠀7<br>
b⠀⠀⠀⠀⠀⠀8<br>
c⠀⠀⠀⠀⠀⠀1<br>
패딩⠀⠀⠀⠀7<br>
<br>
총합   24<br><br>
의 원리로 sizeof(Size)의 결과값을 출력해보면 24가 나온다.<br>
이때 클래스로 객체를 만들면 메모리에 들어가는 것은 멤버 변수지, 멤버 함수가 아니기에 멤버 함수는 sizeof 결과에 관여하지 않는다.
