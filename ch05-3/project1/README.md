# ch05-3 실습과제1

## 문제1
참조에 의한 호출의 장점을 2가지 측면(메모리사용량, 처리시간)에서 설명하시오.

## 답
메모리 사용량 측면 - 참조에 의한 호출을 사용하면 메모리 할당이 일어나지 않아 메모리가 절약되는 장점이 있다. <br>
처리 시간 측면 - 참조에 의한 호출을 사용하면 메모리 할당, 초기화, 생성자와 소멸자 실행 등등이 일어나지 않기 때문에 처리할 데이터가 클수록 시간을 더욱 절약할 수 있다. <br>

## 문제2
C++언어 함수의 반환과정을 상세히 설명하라

## 답
임시변수가 생성되고, 반환값으로 초기화된다. 이후 함수 호출문이 임시변수로 치환되어 변수에 대입되고 나면 임시변수는 소멸한다. <br>

## 문제3
반환형이 참조로 선언된 경우 반환값의 의미를 설명하라.

## 답
참조의 경우 같은 공간의 변수를 그저 2개의 이름으로 나타내는 원리이기 때문에 원본 변수 그대로의 값이라고 할 수 있다. <br>

## 문제4
참조 리턴을 응용하는 예제를 인터넷에서 검색하여 실행해보고 설명하시오.
