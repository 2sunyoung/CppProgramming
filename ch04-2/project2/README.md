# ch04-2 실습과제2

## 문제1
매크로상수 NULL이 무엇인지 조사하라, C/C++언어에서 정의의 차이, 정의된 헤더파일, 사용용도 등

## 답 
<b>NULL의 의미:</b> NULL은 포인터가 어떤 객체도 가리키지 않음을 나타내는 매크로 상수로, <br>
C/C++언어에서 정의의 차이: C에서는 (void*)0으로 정의되고 C++에서는 0으로 정의된다. <br>
3. 또한 NULL은 C에서는 <stdio.h>, <stdlib.h>, <stddef.h> 등에, C++에서는 <cstddef>, <cstdio>, <cstdlib> 등의 헤더파일에 정의되어 있다. <br>
4. 주로 포인터 초기화, 메모리 할당 실패 검사, 자료구조에서의 종료 표시 등에 사용된다. <br>

## 문제2
널문자와 NULL의 차이를 설명하시오, 정의, 자료형, 메모리크기, 실제값, 사용용도 등

## 답
객체의 주소를 저장하는 변수이다.<br>

## 문제3
요즘은 NULL 대신에 nullptr을 사용하는데 nullptr은 무엇인지 조사하시오

## 답
배열은 값을 이웃하게 나란히 모아 놓았기 때문에 포인터 연산을 활용하여 배열 각 원소의 주소를 구하면 된다.<br>
우선 포인터에 배열명을 대입하면 포인터 변수는 배열의 시작 주소를 가리키게 되고, <br>
여기에 인덱스에 해당하는 값을 더하면 그 위치로 포인터가 이동하게 된다.<br>
즉, p + i는 arr[i]의 주소와 같다.<br>
