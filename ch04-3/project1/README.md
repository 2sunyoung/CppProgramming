# ch04-3 실습과제1

## 문제1
메모리 누수를 설명하라.

## 답
동적 할당 후 더 이상 사용하지 않는 메모리를 해제하지 않아 힙 영역에 메모리가 계속 쌓이는 현상이다.<br>

## 문제2
메모리 누수가 왜 심각한 문제를 발생시키는지 설명하라.

## 답
힙 메모리가 점점 부족해져서 다른 프로그램이 메모리를 할당받지 못할 수 있으며, 프로그램의 성능 저하나 강제 종료와 같은 문제가 발생한다.<br>

## 문제3
메모리 누수를 방지하는 방법을 설명하라.

## 답
동적 할당한 메모리를 반드시 해제하거나 스마트 포인터를 사용한다.<br>

## 문제4
C++ 스마트 포인터 클래스에 대해 조사하라.

## 답
C++ 스마트 포인터는 동적으로 할당한 메모리를 자동으로 해제하여 메모리 누수 방지를 목적으로 사용하는 클래스이다.<br>
