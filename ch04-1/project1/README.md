# ch04-1 실습과제1

## 문제1
배열명의 의미를 설명하라.

## 답 
배열의 첫번째 원소의 주소이다. <br>

## 문제2
객체 포인터의 의미를 설명하라.

## 답
객체의 주소를 저장하는 변수이다.<br>

## 문제3
포인터를 이용하여 배열 각 원소의 주소를 구하는 방법을 설명하라.

## 답
배열은 값을 이웃하게 나란히 모아 놓았기 때문에 포인터 연산을 활용하여 배열 각 원소의 주소를 구하면 된다.<br>
우선 포인터에 배열명을 대입하면 포인터 변수는 배열의 시작 주소를 가리키게 되고, <br>
여기에 인덱스에 해당하는 값을 더하면 그 위치로 포인터가 이동하게 된다.<br>
즉, p + i는 arr[i]의 주소와 같다.<br>

## 문제4
포인터를 이용하여 배열 각 원소의 값을 구하는 방법을 설명하라.

## 답
문제3에서 언급한 포인터 연산을 사용하여 배열의 각 원소 주소에 접근한 뒤,<br>
간접 참조 연산자(*)를 이용하여 값을 구하면 된다.<br>
즉, *(p + i)는 arr[i]와 같다.<br>
