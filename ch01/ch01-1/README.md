# ch01-1 실습과제1

## 문제1
클래스와 객체가 무엇인지 설명하시오.

## 답
클래스는 사용자 정의 자료형이고, 객체는 클래스형 변수이다.<br>
클래스는 사용자가 임의로 유사한 속성을 가진 변수나 함수등을 모아 정의하는 자료형이기 때문에 c언어의 구조체와 비슷한 느낌이고, 
객체는 이러한 클래스라는 자료형에서 생겨난 변수이다.

## 문제2
절차지향 프로그래밍 언어와 객체지향 프로그래밍 언어의 차이를 조사하라.

## 답
절차지향 프로그래밍 언어는 프로그램을 처리 절차와 함수 중심으로 구성하며, 실행 순서에 따라 명령을 수행하는 방식이다. 빠르고 직관적이지만 유지 보수와 확장성이 어렵다는 특징이 있다.<br>
객체지향 프로그래밍 언어는 프로그램을 객체 중심으로 구성하여 데이터와 함수를 하나의 객체로 묶어 객체 간의 상호작용으로 프로그램을 작성하는 방식이다. 구조적이고 유지 보수가 편하지만 설계가 복잡하다.

## 문제3
라이브러리가 무엇인지 자세히 설명하시오.

## 답
라이브러리란 프로그램을 만들 때 많이 사용되는 기능을 미리 작성해 놓은 코드로, 목적파일 형태로 존재한다. 예시로 printf, scanf, cin, cout 등이 있다.

## 문제4
프로그램 개발과정 편집->컴파일->링크->실행 단계를 자세히 설명하시오.

## 답
편집: 소스 코드를 작성하는 단계<br>
컴파일: 소스 코드의 문법을 체크하고 기계어(이진수)로 번역하는 단계<br>
링크: 목적 파일과 c++ 표준 라이브러리 파일(목적 파일)을 연결하여 하나의 실행 파일을 만드는 단계<br>
실행: 실행파일의 내용을 컴퓨터가 수행하는 단계
