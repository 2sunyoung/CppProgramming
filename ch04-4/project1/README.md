# ch04-4 실습과제1

## 문제1
string 객체는 문자열 입력 받을 때 몇 자를 입력할지 물어보지않고 알아서 크기를 자동 조절한다. <br>
동작 원리를 검색해보고 설명하라

## 답
C++의 string 객체는 내부적으로 동적 메모리를 사용하여 문자열을 저장하므로, 입력 길이를 미리 지정할 필요가 없다. <br>
문자열이 추가될 때 현재 공간이 부족하면 더 큰 메모리를 새로 할당하고 기존 데이터를 복사하여 크기를 자동으로 확장한다. <br>
또한 효율성을 위해 한 번에 더 큰 용량을 확보하는 방식으로 동작한다.<br>

## 문제2
문자열을 입력할 때 마지막에 입력하는 엔터키와 getline 함수의 마지막 인자에 전달하는 구분문자와 차이를 설명하라

## 답
엔터키는 입력이 끝났음을 나타내며 입력 스트림에 개행 문자(\n)로 저장된다. 반면 getline 함수의 구분문자는 어디까지 문자열을 읽을지를 결정하는 기준이다. <br>
기본적으로 \n을 구분문자로 사용해 엔터 입력 시 읽기를 멈추며, 다른 문자를 지정하면 그 문자가 나올 때까지 입력을 계속 받는다.<br>
