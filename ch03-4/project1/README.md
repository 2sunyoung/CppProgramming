# ch03-4 실습과제1

## 문제1
접근지정자가 왜 필요한가?

## 답
외부로부터 잘못된 접근을 막아 데이터를 보호하기 위한 목적으로 필요하다.<br>

## 문제2
생성자와 setter 멤버함수는 모두 멤버변수를 설정하는 기능이다. 차이점을 설명하라.

## 답
생성자는 객체 생성시 멤버 변수의 초기화 작업을 수행하는 함수이고, <br>
setter 멤버 함수는 객체 생성 이후에 멤버 변수에 접근하여 값을 설정하는 용도로 사용된다.<br>

## 문제3
객체지향형 언어의 특징인 캡슐화와 정보은닉을 설명하라.

## 답
캡슐화란 데이터와 함수를 클래스라는 캡슐 안에 모아 놓고 외부로부터 안전하게 보호하는 기능이고,<br>
정보은닉이란 멤버 변수를 private으로 선언하고 해당 변수를 접근하는 함수(setter(), getter())를 별도로 정의해서 안전한 형태로 멤버 변수에 접근할 수 있도록 하는 기능이다.<br>
