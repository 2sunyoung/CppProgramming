# ch05-4 실습과제1

## 문제1
일반생성자와 복사생성자의 차이는?

## 답
일반생성자는 객체가 생성될 때 주로 초기화를 진행하는 함수이고, <br>
복사생성자는 이미 존재하고 있는 같은 자료형의 객체의 값을 이용하여 새로운 객체를 초기화하는 함수이다.<br>

## 문제2
대입연산과 얕은 복사생성자의 차이는 무엇인가?

## 답
대입연산은 기존에 있던 객체의 값을 덮어쓰는 연산이고, 얕은 복사 생성자는 객체가 생성될 때 같은 자료형의 객체를 이용하여 단순 초기화를 는 함수이다. <br>
이때 얕은 복사생성자는 새로운 메모리 공간을 따로 할당하지 않고 단순 대입만 하는 특징이 있다. <br>

## 문제3
지금까지 배운 내용기준으로 컴파일러가 자동으로 추가하는 멤버함수는 어떤 것이 있는가?(4가지)

## 답
생성자, 복사생성자, 디폴트 생성자, 소멸자 등이 있다.<br>
