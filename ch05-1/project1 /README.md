# ch05-1 실습과제1

## 문제1
인자와 매개변수를 구분하여 설명하시오.

## 답
인자란 함수를 호출할 때 매개변수에 전달하는 값으로, 호출하는 함수의 지역변수로 표현할 수 있다.<br>
매개변수란 함수를 정의할 때 함수명 옆에 선언하며 인자의 값을 전달받는 일반 변수로, 호출 당하는 함수의 지역변수로 표현할 수 있다.<br>

## 문제2
함수 호출시 자동으로 처리되는 2가지는 무엇인가?

## 답
함수가 호출되면 ① 매개변수가 메모리에 할당되고, ② 매개변수가 인자값으로 초기화된다.

## 문제3
값에 의한 호출로 다른 함수에서 선언된 변수의 값을 변경할 수 없다. 이유를 설명하라.

## 답
지역변수는 해당 함수에서만 생성되고 소멸되는 변수이다. 값에 의한 호출에서는 매개변수에 인자값이 복사되는 구조이므로, <br>
호출된 함수 내부에서 값을 변경하는 작업을 하더라도<br>
지역변수인 매개변수에만 영향을 줄 뿐 다른 함수의 원래 변수에는 영향을 주지 않는다. <br>
따라서 값에 의한 호출로 다른 함수의 지역변수를 변경하는 것은 불가능하다.<br>

## 문제4
주소에 의한 호출을 사용해야하는 경우를 2가지 이상 설명하라.

## 답
1. 호출된 함수에서 선언된 지역 변수의 값을 변경한 결과를 다른 함수에서도 그대로 써야하는 경우<br>
2. 배열의 데이터를 전달받아야 하는 경우<br>
3. 2개 이상의 값을 리턴해아하는 경우<br>

## 문제5
인자가 객체인 경우는 어떤 호출방식이 좋은가? 이유도 설명하라.

## 답
인자가 객체인 경우는 주소에 의한 호출 방식이 좋다. <br>
왜냐하면 주소에 의한 호출로 함수에 객체를 전달할 때 함수 호출시 인자로 객체의 주소를 전달하고, 매개변수는 객체가 아닌 '객체 포인터'로 선언하기 때문에 <br>
생성자와 소멸자가 실행되지 않고, 주소만 복사하면 되므로 객체를 복사하는데 걸리는 시간과 메모리를 절약할 수 있다는 장점이 있기 때문이다.<br>

## 문제6
함수 반환과정에서 자동으로 처리되는 2가지는 무엇인가?

## 답
① 임시변수가 할당된 다음 반환값으로 초기화, ② 함수 호출 문장이 임시변수(반환값)로 치환, 대입 후 임시변수 소멸

## 문제7
함수 반환과정에서 임시변수가 필요한 이유를 설명하라.

## 답
반환값은 함수 안에서 선언된 지역변수이므로 함수 반환 후 메모리에서 사라진다. <br>
이때 반환값이 소멸되기 전에 임시변수라는 것을 만들어 안전하게 백업해두는 과정이 필요하다. <br>
따라서 이러한 과정을 정상적으로 수행하기 위해 임시변수가 필요한 것이다.<br>
