# ch07-1 실습과제1

## 문제1
프렌드 함수가 왜 필요한가?

## 답
클래스의 멤버함수로 적합하지 않지만, 클래스의 모든 멤버를 자유롭게 접근해야 하는 경우 프렌드 함수가 필요할 수 있다.<br>
가령 연산자 함수를 클래스의 밖에서 정의했다면, <br>
클래스 안의 모든 멤버에 접근하기 위해 프렌드 키워드를 사용하여 기존의 private 멤버 접근 불가 문제를 해결할 수 있다.<br>

## 문제2
객체 지향형 언어의 특징 중의 하나인 다형성의 의미와 종류를 조사하시오.

## 답
다형성이란, 하나의 객체나 함수가 상황에 따라 여러 가지 서로 다른 형태로 동작하는 성질을 의미한다.<br>
대표적으로 같은 이름의 함수를 매개변수의 개수나 자료형을 다르게 하여 여러 개 정의하는 방식인 <b>함수 오버로딩</b>과<br>
기존 연산자에 새로운 기능을 정의하여 사용자 정의 객체에도 연산자를 사용할 수 있게 하는 방식인 <b>연산자 오버로딩</b> 등이 있다.<br>
