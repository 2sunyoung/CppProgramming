# 소스코드 설명

```
#include <iostream>
```

- cin, cout 등 라이브러리 객체의 선언을 포함하고 있는 헤더파일 iostream을 포함한다.

```
using namespace std;
```

- std 이름 공간에 선언된 모든 이름에 std::생략

```
int main(void)
```

- 메인함수 시작

```
cout << "군산대학교 \"홍길동\"" << endl;
```

- "군산대학교 “홍길동”" 출력, 줄바꿈

```
cout << "\"축하합니다.\"" << endl;
```

- "“축하합니다.”" 출력, 줄바꿈

```
cout << "100% \\취업율\\";
```

- "100% \취업율\" 출력, 줄바꿈

```
return 0;
```

- 0을 반환하고 메인함수 정상 종료
