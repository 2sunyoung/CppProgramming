# ch03-3 실습과제1

## 문제1
객체의 생성과 소멸의 의미를 설명하라.

## 답
객체의 생성이란 운영체제가 관리하는 메모리 공간을 프로그램에서 사용할 수 있게 됨(=할당 받음)을 의미한다.<br>
객체의 소멸이란 프로그램에서 사용하던 메모리 공간을 운영체제에 반납하고 더 이상 그 공간을 사용할 수 없게 됨을 의미한다.<br>

## 문제2
지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라, 선언위치, 접근범위, 생성시점, 소멸시점, 초기화 여부 등의 관점에서 비교하라.

## 문제3
프로그램 실행 후 main함수가 실행되기 전에 어떤 일이 처리되는지 조사해보라.

## 답
전역 변수와 static 변수가 0으로 자동 초기화된다.<br>
만약 전역 객체가 있다면 main 함수 실행전에 생성이 되기 때문에 생성자도 객체 생성과 동시 호출된다.<br>

## 문제4
main함수가 종료 후 프로그램이 종료하기 전에 어떤 일이 처리되는지 조사하라.

## 답
main함수 안에서 생성된 지역객체(변수)들이 소멸된 후 main 함수 밖에서 생성된 전역객체(변수)들이 차례로 소멸된다.<br>
